## Purpose

This scoring card turns the **Digital Age Antique Value Model** into a practical observation tool.

It helps evaluate whether a real physical object may carry future collectible meaning.

This card is not designed to predict exact resale price.

It is designed to support human judgment.

## Scoring Method

Each signal is scored from **0 to 5**.

| Score | Meaning |
|---|---|
| 0 | No clear signal |
| 1 | Very weak signal |
| 2 | Weak but visible signal |
| 3 | Moderate signal |
| 4 | Strong signal |
| 5 | Very strong signal |

The final score does not guarantee value.

It only shows how strong the object appears as an observation candidate.

---

## Value Signal Scorecard

| Signal | Score 0–5 | Notes |
|---|---:|---|
| Cultural Memory |  | Does this object connect to a person, brand, era, technology, lifestyle, or shared memory? |
| Physical Authenticity |  | Is the object real, material, original, and not only a reproduction or digital image? |
| Scarcity |  | Is it becoming harder to find in clean, complete, working, or boxed condition? |
| Condition |  | Is the object preserved, visually strong, working, and free from major damage? |
| Completeness |  | Are box, manual, accessories, labels, stickers, inserts, receipts, or original parts included? |
| Provenance |  | Can the origin, ownership history, purchase history, or context be explained? |
| Display Value |  | Can the object create meaning in a home, studio, store, museum, or visual space? |
| Story Value |  | Does the object tell a story beyond its original function? |
| Buyer Demand |  | Do collectors, fans, designers, creators, or ordinary people still care about this object? |
| Human Judgment Confidence |  | After reviewing all signals, how confident is the observation? |

| 36–45 | Strong observation candidate |
| 46–50 | High-interest object, still not guaranteed |

---

## Decision Layer

After scoring, choose one decision category:

| Decision | Meaning |
|---|---|
| Avoid | Too damaged, too incomplete, too common, fake, overpriced, or unclear. |
| Watch | Interesting, but more research is needed. |
| Buy Low | Worth considering only if the price is low enough to reduce risk. |
| Hold | Already owned or bought low; may be worth preserving. |
| Document | Worth recording as cultural evidence even if not purchased. |
| Sell | May be worth selling if demand is visible and condition is not strong enough to hold. |

---

## Red Flags

Be careful if the object has:

- missing parts
- missing original packaging
- no clear photos
- vague seller description
- mold
- water damage
- battery swelling
- fake labels
- reproduction packaging
- too much hype
- price based only on fame
- no sold-price comparison
- poor storage condition
- unclear authenticity
- seller pressure
- “rare” claim without evidence

---

## Stronger Signals

An object becomes stronger when it has:

- complete packaging
- clean condition
- visible brand or artist connection
- strong cultural memory
- clear production era
- original manuals or accessories
- working function
- display appeal
- emotional recognition
- documented ownership
- low purchase price compared with risk
- increasing public nostalgia
- connection to early digital life or pre-AI material culture

## Example Reading

A first-generation digital device with original box, clean condition, accessories, and strong cultural memory may score high.

A famous artist cassette with damaged packaging may still carry cultural memory, but condition and completeness must reduce the score.

A common object with no story, no packaging, and no buyer demand should not be judged valuable only because it is old.

---

## Human Judgment Note

The score is not the final answer.

The score only slows down the decision.

The final judgment should ask:

**Does this object still carry enough real-world meaning, authenticity, condition, and story to matter later?**

An object can be old and still weak.

An object can be common today and become meaningful later.

The difference is not only age.

The difference is structure.
