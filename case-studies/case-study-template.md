## What Usually Weakens Value

This object may have weaker future value if it is:

- broken
- incomplete
- fake or reproduced
- missing original packaging
- heavily damaged
- poorly stored
- too common
- disconnected from cultural memory
- impossible to verify
- purchased at an inflated price
